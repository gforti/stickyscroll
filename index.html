Stickyscroll

A JavaScript class that allows for a sidebar to follow the user as they scroll down the page.

View project onGitHub

This assumes that the side bar is on the right and not on the left. The code could be adjusted to make it for configurable but for now it is not supported.

Demo 1 | Demo 2 | Demo 3

@params

  • scrollerID - The ID of the Div to set as the sticky scroll object.
  • rightOfID - The ID of the Div to set the location of the sticky scroll object to the right off.
  • limitID - The ID of the Div to set the scroll bottom limit of the sticky scroll object.

Usage

stickyScroll.observe( scrollerID, rightOfID [,limitID] );

e.g:

stickyScroll.observe("sidebar","content");

Code

The code is constructed as a JavaScript singleton. You can set multiple div's to be a sticky scroll object.
